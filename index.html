jQuery Notebook Plugin Example

Notebook is a jQuery plugin for creating a floating toolbar containing an multi-line or inline WYSIWYG rich text editor that allows you to in-place edit the selected text/block of the page. The plugin currently works in modern browsers which support CSS3 transform property.
